# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.1] - 2026-04-23

### Changed
- Upgrade pvbt dependency to v0.7.7

## [0.1.0] - 2026-04-21

### Added
- Initial release of Bold Asset Allocation strategy
- Canary universe crash-protection signal that shifts allocation between offensive and defensive asset universes based on absolute momentum
- Configurable top-N selection from offensive and defensive universes, supporting BAA-G4, BAA-G12, and trimmed variants
